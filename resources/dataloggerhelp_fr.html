Configuration du logiciel

Les paramètres du logiciel:

Taille du buffer : nombre de mesures effectuées (par défaut 1200).
Cette valeur peut être adaptée selon les besoins de l'utilisateur.

Titre du graphique : donnez un titre explicite à vos données (ex: Hygrométrie le 14/07/2020)
Ce titre sera enregistré si vous sauvegardez les données.

Vitesse d'echantillonage : sélectionnez l'unité (millisecondes, secondes ou minutes) puis une valeur.
Cette valeur, en relation avec la taille du buffer, va déterminer le temps d'enregistrement.
Ainsi avec une lecture par seconde et un buffer de 1200 points nous avons (1x1200/60) 20 minutes d'enregistrement.

Nom et unité des données : Ceci est affiché sur l'axe y du graphique.
Si une unité SI est fournie, le graphique utilisera automatiquement les multiples.

Min et max de l'axe Y : Ces paramètres permettent de régler la plage d'affichage du graphique.

Couleur du graphique : Un clic sur le bouton fera apparaître un sélecteur de couleurs.
Vous pourrez choisir la couleur de la ligne du graphique.

Utiliser le filtre : Cochez ou décochez cette case pour utiliser ou non le filtre interne.
Si cette case est décochée les données brutes issues du port série seront utilisées.

Langues : Sélectionnez la langue de l'interface.


Cliquez 'Enregistrer' pour sauvegarder ces paramètres comme paramètres par défaut.

Cliquez 'Ok' pour utiliser ces paramètres pendant la session courante uniquement.


Les paramètres du port série:

Port série : Sélectionnez dans la liste le port à utiliser.
Si la carte arduino a été branchée recement, utilisez le bouton 'scan' pour rechercher de nouveaux ports.

Baud rate : Sélectionnez la bonne vitesse de connexion (57600 par défaut).

Bits de données : Sélectionnez la bonne valeur ((8 par défaut).)

Bits de parité et d'arrêt : Sélectionnez les bonnes valeurs.

Autres : La boite de dialogue permet de spécifier d'autres paramètres.
Ceux-ci ne sont pas utilisés par le logiciel et on peut laisser les valeurs par défaut.


Cliquez 'Ok' pour confirmer les modifications


Éditer le filtre de traitement:

Le filtre de traitement est une fonction python destinée à traiter les données reçues du port série.
Celle-ci peut être éditée dans le logiciel via un éditeur intégré ou en éditant le fichier tools/datafilters.py.
Par défaut cette fonction convertit une valeur de l'ADC arduino (0 - 1023) en une tension (0 - 5V) mais pourra effectuer n'importe quel traitement.
Bien sûr il faudra qu'il s'agisse de code python valide sous peine de faire planter le logiciel.

Réaliser des mesures

Les donnés de la carte arduino:

La carte arduino (ou toute carte disposant d'une communication série) peut mesurer n'importe quel type de capteur (tension, température etc). Toutefois, la valeur envoyée par le port série doit être un float,seul type reconnu par le logiciel.

Cette valeur pourra être traitée par la carte et ne pas utiliser le filtre interne ou fournir une valeur brute traitée ensuite par le logiciel.


Lancer une mesure:

Pour effectuer une acquisition depuis la carte arduino :

  1. Cliquer 'Conf. série' ou Menu->Port série->Configurer.
    Une fois le port correctement configuré, le bouton 'Connecter' est disponible.

  2. Cliquer 'Connecter' ou Menu->Port série->Connecter.
    Le logiciel se connecte à la carte arduino. Le bouton 'Démarrer' est disponible.

  3. Cliquer 'Démarrer'. On vous demandera si vous voulez effacer le buffer.
    Le bouton devient 'Stop' et vous permet d’arrêter l'acquisition des données.
    Vous pouvez cliquer à nouveau sur 'Démarrer' et reprendre la lecture (sans effacer le buffer)
    ou recommencer l'acquisition en effaçant le buffer.
    L'acquisition s'arrête automatiquement lorsque le buffer est plein.


Les fichiers sauvegardés :

Enregistrer un fichier :Lorsque le logiciel a démarré la lecture vous pouvez à tout moment
enregistrer les données dans un fichier pour une analyse ultérieure en cliquant sur le bouton 'Enregistrer'.


Charger un fichier : Lorsque le logiciel n'est pas en mode acquisition, vous pouvez charger un fichier
enregistré en cliquant sur 'Ouvrir' ou Menu->Fichier->Ouvrir. Les données sont alors affichées sur le graphique.
En utilisant la souris vous pourrez zoomer et vous déplacer dans le graphique (ceci est vrai aussi pendant l'acquisition!).
Un clic droit sur le graphique ouvre un menu vous permettant entre autres d'enregistrer une image du graphique (export).
